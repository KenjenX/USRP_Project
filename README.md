# USRP B210 Scanner

Proyek monitoring spectrum realtime menggunakan USRP B210, Python, UHD, dan web dashboard.

## Fitur Saat Ini
- Membaca IQ sample dari USRP B210
- Menampilkan realtime spectrum
- Peak frequency detection
- Threshold monitoring
- Warning log ke JSON

## Rencana Pengembangan
- Web dashboard
- Deteksi beberapa peak
- Klasifikasi band GSM
- Integrasi data realtime ke browser
